# Encrypted Chat

It is a socket programming based crypto live messaging application.
## Installation

Use the package manager [npm](https://nodejs.dev/) to install packages.

```bash
npm install
```

## Usage

```bash
npm run devStart
```
Socket Server is started.
You can access in index.html by your browser.
